## Overview

The Home Assistant Text Sensor platform imports text and string values from Home Assistant entities, allowing ESPHome devices to display weather descriptions, status messages, and other text data.

## Related Devices

- [JC2432W328C ESP32 2.8" Display](/devices/jc2432w328c) - Used to display weather descriptions and climate status
- [DFRobot Beetle ESP32-C6](/devices/beetle-esp32-c6) - Used to display temperature units, weather conditions, and preset modes
